# Alibaba-Iconfont-downloads

阿里巴巴矢量图标库 `图标` 一键下载脚本

## 📖 使用方法

### 🔖 书签执行

1. 创建书签, 填写名称, 复制[书签版本](./dist/bookmark.txt)代码到书签网址上, 注意去掉顶部注释!!!
    
![创建书签](./image/create.png)

2. 去图标库详情页面点击标签即可

### 🏃 F12执行

1. 打开图标库详情页

2. F12打开控制台, 将[控制台版本](./dist/console.js)代码粘贴并执行即可
